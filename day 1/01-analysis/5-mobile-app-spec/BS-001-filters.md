# Требования на дизайн · BS-001 · Фильтры

**ID:** BS-001 · **Тип:** Bottom Sheet · **Зона:** АЗ · **Приоритет:** High · **Статус:** Черновик · **Версия:** 0.1 · **Дата:** 2026-07-04

**Источники:** [Функциональные требования](../2-requirements/functional-requirements.md) · [Use cases](../2-requirements/use-cases.md) · [User stories](../2-requirements/user-stories.md)

## 1. Назначение и контекст

Шторка фильтров уточняет список слотов без перехода на другой экран. Она используется вместе со списком слотов и должна давать быстрый контроль над выдачей.

## 2. Структура и иерархия

- Фильтр по датам.
- Фильтр по типу слота или занятия.
- Фильтр «только со свободными местами».
- Фильтр по мастеру.
- Кнопки «Применить» и «Сбросить».

## 3. Состояния

| Состояние | Поведение |
|-----------|-----------|
| Default | Фильтры не применены |
| Filled | Выбраны значения фильтров |
| Empty result | После применения фильтров список на основном экране может стать пустым |

## 4. Входные данные

| Название | Тип | Возможные значения | Описание |
|----------|-----|-------------------|----------|
| `date_from` | Состояние | datetime | Начало диапазона |
| `date_to` | Состояние | datetime | Конец диапазона |
| `route_type` | Состояние | `novice`, `experienced` | Тип маршрута |
| `instructor_id` | Состояние | UUID | Мастер |
| `only_available` | Состояние | `true` / `false` | Только слоты со свободными местами |

## 5. Поведение

- Фильтры комбинируются между собой.
- При нажатии «Применить» список слотов обновляется.
- При нажатии «Сбросить» все параметры возвращаются к значениям по умолчанию.
- Если по фильтрам нет результатов, пользователь должен увидеть пустое состояние на экране списка.

## 7. Критерии приемки

| ID | Критерий |
|----|----------|
| AC-001 | **Дано** пользователь открыл шторку фильтров, **Когда** он выбирает параметры и нажимает «Применить», **Тогда** список слотов обновляется |
| AC-002 | **Дано** фильтры уже выбраны, **Когда** пользователь нажимает «Сбросить», **Тогда** шторка возвращается к состоянию по умолчанию |
