# Требования на дизайн · SCR-003 · Карточка слота

**ID:** SCR-003 · **Тип:** Экран · **Зона:** АЗ · **Приоритет:** Critical · **Статус:** Черновик · **Версия:** 0.1 · **Дата:** 2026-07-04

**Источники:** [Функциональные требования](../2-requirements/functional-requirements.md) · [Use cases](../2-requirements/use-cases.md) · [User stories](../2-requirements/user-stories.md)

## 1. Назначение и контекст

Экран показывает подробности конкретного слота до записи. Это промежуточная точка между списком и оформлением бронирования.

## 2. Навигация

### Входящие переходы
- Тап по карточке в [SCR-002-slot-list.md](SCR-002-slot-list.md).

### Исходящие переходы
- Тап по кнопке записи → [SCR-004-booking.md](SCR-004-booking.md).
- Тап по карте маршрута → [BS-004-route-map.md](BS-004-route-map.md).

## 3. Входные данные

| Название | Тип | Возможные значения | Описание |
|----------|-----|-------------------|----------|
| `slot_id` | Параметр перехода | UUID | Идентификатор слота |
| `slot` | Ответ API | Slot | Полные данные слота |

## 4. Используемые запросы

### GET /slots/{slotId}

**Триггер:** открытие экрана.

**Назначение:** получить полные данные слота, включая маршрут и точку встречи.

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| `200` | Показать полную карточку слота |
| `404` | Показать, что слот больше недоступен |
| `401` | Вернуть на вход |
| `5xx` | Показать ошибку загрузки |

## 5. Состояния экрана

| Состояние | Поведение |
|-----------|-----------|
| Loading | Загружается карточка слота |
| Content | Показаны все данные слота |
| Error | Сообщение о невозможности загрузить данные |

## 6. Поведение

- Пользователь должен видеть цену до перехода к оформлению записи.
- Если свободных мест нет, действие записи недоступно.
- Экран должен вести к экрану оформления записи.

## 8. Критерии приемки

| ID | Критерий |
|----|----------|
| AC-001 | **Дано** клиент открыл карточку слота, **Когда** данные загружены, **Тогда** видны дата, время, мастер, цена и свободные места |
| AC-002 | **Дано** в слоте нет свободных мест, **Когда** экран отображается, **Тогда** кнопка записи недоступна |
