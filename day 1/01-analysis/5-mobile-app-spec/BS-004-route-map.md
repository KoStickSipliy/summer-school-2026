# Требования на дизайн · BS-004 · Карта маршрута

**ID:** BS-004 · **Тип:** Bottom Sheet · **Зона:** АЗ · **Приоритет:** Medium · **Статус:** Черновик · **Версия:** 0.1 · **Дата:** 2026-07-04

**Источники:** [Функциональные требования](../2-requirements/functional-requirements.md) · [Use cases](../2-requirements/use-cases.md)

## 1. Назначение и контекст

Шторка показывает карту маршрута и помогает клиенту сориентироваться по точке проведения занятия.

## 2. Состав контента

- Карта маршрута.
- Точка встречи.
- Визуальное выделение маршрута.
- Закрытие шторки.

## 3. Поведение

- Открывается из карточки слота или деталей брони.
- Показывает актуальную географическую привязку занятия.
- Не должна мешать основному сценарию записи и отмены.

## 4. Входные данные

| Название | Тип | Возможные значения | Описание |
|----------|-----|-------------------|----------|
| `route.geometry` | Данные слота | array / encoded polyline | Линия маршрута |
| `meeting_point` | Данные слота | string | Точка встречи |
| `meeting_point_lat` | Данные слота | float | Широта точки встречи |
| `meeting_point_lng` | Данные слота | float | Долгота точки встречи |
