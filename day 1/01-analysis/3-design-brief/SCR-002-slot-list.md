## 1. Назначение и контекст

SCR-002 — главный экран приложения и стартовая вкладка авторизованной зоны. На экране показывается список доступных слотов на ближайшие 7 дней, а также вход в сценарий фильтрации и просмотра подробностей слота.

---

## 2. Пользователь и цель

**Роль — Клиент.**

**Цель на экране:** быстро найти подходящее занятие и перейти к записи.

---

## 3. Навигация

### Входящие переходы
- После успешного входа с [SCR-001](SCR-001-registration.md).
- По тапу на таб «Слоты» из любого корневого экрана.
- Возврат из [SCR-003](SCR-003-slot-card.md) и [BS-001](BS-001-filters.md).

### Исходящие переходы
- Тап по карточке слота → [SCR-003](SCR-003-slot-card.md).
- Тап по «Фильтры» → [BS-001](BS-001-filters.md).
- Тап по табу «Мои бронирования» → [SCR-005](SCR-005-my-bookings.md).
- Тап по табу «Профиль» → [SCR-007](SCR-007-profile.md).

---

## 4. Структура и иерархия

- Хедер с названием экрана.
- Кнопка «Фильтры» с индикатором активных фильтров.
- Список карточек слотов.
- Таб-бар внизу экрана.

Карточка слота должна быть единой тап-зоной. В ней показываются дата и время, название занятия или маршрута, мастер, цена и информация о свободных местах.

---

## 5. Состояния экрана

| Состояние | Содержимое / поведение |
|-----------|------------------------|
| Loading | Скелетоны карточек слотов |
| Content | Список доступных слотов, отсортированный по ближайшей дате |
| Empty | Понятная заглушка без доступных слотов |
| Empty by filters | Заглушка с подсказкой изменить или сбросить фильтры |
| Error | Сообщение об ошибке и кнопка обновления |

---

## 6. Фильтры и поведение списка

- По умолчанию показываются слоты на ближайшие 7 дней.
- На экране должен быть вход в [BS-001](BS-001-filters.md).
- При применении фильтров показывается индикатор активных фильтров.
- Список можно обновить pull-to-refresh.
- Слоты без свободных мест отображаются как недоступные для записи.

---

## 7. Состав контента

| Элемент | Назначение |
|---------|------------|
| Дата и время | Основной ориентир при выборе слота |
| Название занятия | Понимание, что это за слот |
| Мастер | Контекст занятия |
| Цена | Стоимость до перехода к записи |
| Свободные места | Быстрая оценка доступности |

## 9. Критерии приемки

```gherkin
AC-001: Базовый список
  Дано клиент открыл экран списка слотов
  Когда данные загружены
  Тогда видны слоты на ближайшие 7 дней

AC-002: Индикатор фильтров
  Дано применен хотя бы один фильтр
  Когда список показан
  Тогда в хедере виден индикатор активных фильтров

AC-003: Пустой результат
  Дано по фильтрам нет слотов
  Когда список обновлен
  Тогда показана заглушка с подсказкой изменить или сбросить фильтры
```
