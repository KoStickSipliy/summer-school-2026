## 1. Назначение и контекст

Шторка фильтров уточняет список слотов без перехода на другой экран. Она используется вместе со списком слотов и должна давать быстрый, предсказуемый контроль над выдачей.

---

## 2. Структура и иерархия

- Фильтр по датам.
- Фильтр по типу слота или занятия.
- Фильтр «только со свободными местами».
- Фильтр по мастеру.
- Кнопки «Применить» и «Сбросить».

---

## 3. Состояния

| Состояние | Поведение |
|-----------|-----------|
| Default | Фильтры не применены |
| Filled | Выбраны значения фильтров |
| Empty result | После применения фильтров список на основном экране может стать пустым |

---

## 4. Правила фильтрации

- Фильтры комбинируются между собой.
- При нажатии «Применить» список слотов обновляется.
- При нажатии «Сбросить» все параметры возвращаются к значениям по умолчанию.
- Если по фильтрам нет результатов, пользователь должен увидеть пустое состояние на экране списка.

## 6. Критерии приемки

```gherkin
AC-001: Применение фильтра
  Дано пользователь открыл шторку фильтров
  Когда он выбирает параметры и нажимает «Применить»
  Тогда список слотов обновляется

AC-002: Сброс фильтров
  Дано фильтры уже выбраны
  Когда пользователь нажимает «Сбросить»
  Тогда шторка возвращается к состоянию по умолчанию
```
